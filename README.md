### Install: 
```bash
git clone "https://github.com/diegoara96/createRepoAutomat.git"
cd createRepoAutomat
pip install -r requirements.txt
source .createrepo.sh
Then go to createrepo.py and put your github tocken
all projects are going to be saved in ~/Documentos/proyectos change if you want in .createrepo.sh
```

### Usage:
```bash
To run the script type in 'createrepo <name of your folder>'
```


### Note:
```bash
if you want to be able to execute at any time rename the .createrepo.sh something like createrepo (whitout .sh) 
and copy to path
cp .createrepo.sh /usr/local/bin/createrepo
```
